# Data Analysis Exercises

A collection of beginner-friendly Python exercises focused on data analysis using Pandas, NumPy, Matplotlib, and Seaborn.

## Project Structure

- `exercise1_basic_analysis.py`  
  Performs product sales analysis, calculates total revenue, identifies the best-selling product in each category, and creates a revenue bar chart.

- `exercise2_time_series.py`  
  Creates one month of daily sales data, calculates a 7-day moving average, identifies the weekday with the highest average sales, and visualizes the results.

- `exercise3_data_cleaning.py`  
  Demonstrates a complete data-cleaning workflow by removing duplicates, handling missing values, correcting data types, and standardizing text values.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
